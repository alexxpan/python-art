arrow
arrow(x, y, width, type=NORMAL, draw=True)
Draws an arrow to the screen. The first two parameters set the location (measured from the arrow's head), the third sets the width. Optionally you can define the arrow type with the fourth parameter. Default the arrow type is NORMAL, but there is also a trendier FORTYFIVE version.
 
arrow(50, 50, 50)

a path containing the arrow
