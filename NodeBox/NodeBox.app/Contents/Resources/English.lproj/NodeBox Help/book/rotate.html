rotate
rotate(degreens=0, radians=0)
Rotates all subsequent shape, path, text, image commands. The rotate() command is called with one parameter, by default specifying the angle in degrees. Optionally, the angle can be specified in radians, by calling rotate(radians=0.5) for example. The rotate() command works incremental: if you call rotate(30), and later on call rotate(60), all commands following that second rotate() will be rotated 90 degrees (30+60).
