size
size(w, h)
Sets the size of the canvas in points. This should be the first line in a script, even before any comments. The first parameters sets the width, the second parameter the height. If this command is used, it should be the first line in a script - it's illogical to set the size of the canvas halfway a script. The WIDTH and HEIGHT constants can be used to obtain the canvas size. A more convenient way to set the canvas size is by using the File > Document Setup menu.
size(400, 600)
# the canvas is 400 X 600 pixels
print WIDTH # prints 400
print HEIGHT # prints 600
