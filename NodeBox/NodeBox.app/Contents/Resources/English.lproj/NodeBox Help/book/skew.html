skew
skew(x, y=None)
Slants the direction of all subsequent shape, path, text, image commands. The first parameter sets the horizontal skew. The second parameter is optional and sets the vertical skew. The skew() command works incremental: if you call skew(10), and later on call skew(20), all commands following that second skew() will be skewed by 30 (10+20).
