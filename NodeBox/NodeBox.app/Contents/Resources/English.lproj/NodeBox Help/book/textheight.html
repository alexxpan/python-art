textheight
textheight(txt, width=None)
Returns what would be the height of a text block drawn with the text() command, in the current font(), fontsize() and lineheight(). The first parameter sets the text string to measure, the second parameter sets the width of a line of text in block.
font("Courier New")
h = textheight("Ekky ekky!", 30)
print h

the height of the text block, as a floating point number
